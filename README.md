# dio-ebook-tecnologia-no-dia-a-dia
E-book criado para o bootcamp da DIO, reunindo conceitos essenciais de tecnologia para iniciantes: fundamentos digitais, organização, produtividade e segurança da informação. Projeto desenvolvido para fortalecer meu portfólio, aprimorar escrita técnica e praticar versionamento com Git e GitHub.
